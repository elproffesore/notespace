# Notespace

This little application is for taking quick notes and edit them via markdown. They are sorted after last edit and can be archived.
This project is a little starting point for further add-ons.

# Start

If you want to use a Nextcloud as a backup-server, update the environment variables in compose.dev.yml and rename ```backup.dev.sh``` to ```backup.sh```
 
---
1. Create a ```database.db``` file in the root directory
2. Run ```docker-compose up --build --detach```
